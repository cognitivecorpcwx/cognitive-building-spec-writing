# Cognitive Building Spec Writing

## Purpose
This skill provides guidance and templates for writing comprehensive specifications tailored for cognitive building initiatives. It helps users ensure clarity and completeness in defining project requirements and functionality.

## When to Use
- When initiating a new cognitive building project.
- When revisions to existing specifications are needed.
- When seeking to standardize specification writing across teams.

## Instructions
1. Identify the project objectives and key outcomes.
2. Gather all relevant stakeholder inputs and requirements.
3. Outline the main sections of the specification document.
4. Write detailed descriptions for each section, including use cases and acceptance criteria.
